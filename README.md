# Customer_Segmentation_Project

This repository contains the code, result csv file, and pdf reports for the customer segmentation project using clustering techniques.

## Files
- **Vignesh_S_Clustering.pdf**: Detailed analysis on Clustering of the customer segmentation and results.
- **Vignesh_S_Clustering.ipynb**: Jupyter Notebook with code and explanations.
- **Vignesh_S_EDA.pdf**: Pre-processing of all the 3 datasets and visualization insights on some trends.
- **Vignesh_S_EDA.ipynb**: Jupyter Notebook with code and explanations.
- **Vignesh_S_Lookalike.csv**: Results of the model built based on the user's information as input and recommending 3 similar
customers based on their profile and transaction history as output.
- **Vignesh_S_Lookalike.ipynb**: Jupyter Notebook with code and explanations for a similar model.


- **data/**: Contains the input datasets `Customers.csv`, `Products.csv` and `Transactions.csv`.

## Instructions
1. Install required Python libraries using:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the code:
   - For Jupyter Notebook: Open `Vignesh_S_EDA.ipynb`,
                                `Vignesh_S_Lookalike.ipynb`
                                `Vignesh_S_Clustering.ipynb`
